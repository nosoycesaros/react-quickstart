# React QuickStart
A really simple quickstart boilerplate for React projects. this project includes:
* React
* Styles File (SCSS)
* React-Router
* Webpack

## Quickstart
1. Install node modules `npm install`
2. Start the server `npm start`

This project is based on the following project list:
